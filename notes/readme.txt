[객체]
oInit
	초기화 용 객체입니다. 무작위 시드값을 설정하고 개체들의 전략 열거형을 정의합니다.

oGameGlobal
	게임 시스템 객체입니다. 메모리 청소, 길찾기 용 격자, UI 표시, 몇몇 변수들의 갱신을 담당합니다.

oHealthIndicator
	인공지능 개체들의 체력을 표시해줍니다.

oShadowDrawer
	인공지능 개체들의 그림자를 표시해줍니다. 단순한 장식 효과용 객체입니다.

oAllySpawner / oEnemySpawner
	각자 가진 인공지능 개체가 죽으면 일정 시간 이후 재생성합니다.

oEntities
	모든 인공지능 개체의 공통적인 행동을 정의한 객체입니다. 이 예제에서 인공지능들은 두 세력으로 나뉘어져 있습니다.
	하지만 세력이 달라도 그 행동은 모두 같은 기반 위에서 작동합니다. 적대적 목표, 아군의 종류 등 특수화가 이루어져야 
	하는 속성들을 제외한 모든 동작들은 이 부모 객체에서 작동합니다. 자세한 내용은 객체의 내용을 참조하세요.

oRed / oBlue
	oEntities를 부모로 가진 인공지능 객체입니다. 자세한 내용은 생성 이벤트를 참조하세요.

oBlock
	Solid(고체) 객체로서 oGameGlobal이 갖고 있는 길찾기 용 격자에 장애물로 등록됩니다.

[룸]
roomInit
	초기화 용 룸입니다. oInit만이 배치되있습니다.

roomGame
	여러 종류의 레이어와 개체들이 배치된 룸입니다. 자세한 내용은 직접 확인하세요.
